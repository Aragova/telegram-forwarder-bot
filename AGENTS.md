# AGENTS.md

## Общие правила работы с проектом

- Это production Telegram forwarder bot.
- Нельзя ломать текущее поведение.
- Все изменения должны быть минимальными и безопасными.
- Не переписывать код без необходимости.

## Архитектурные ограничения

НЕ ТРОГАТЬ без отдельной задачи:

- TransportPolicy
- WorkerPolicy
- VideoProcessor (ffmpeg pipeline)
- reaction logic
- Postgres schema
- audit_log
- problem_state

## Работа с кодом

- Не делать "красивые рефакторинги"
- Не менять структуру проекта без необходимости
